## Pytorch Bert_Atepc 电商评论属性级分析项目

本课程介绍的是属性级情感分析模型，相较于传统的对整个句子的好评或差评的情感分类，本项目先提取评价对象实体，再标注情感分类，将评价对象区分评判，在实际业务中更具有工程意义，该模型常用于电商平台评论分析场景。
